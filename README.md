# Grade Tracker

A desktop grade tracking app where students can view their grades, teachers
can manage assignments and enter scores, and admins can manage users and
courses. Built with JavaFX, SQLite, and AtlantaFX for UI.

## Team
- Olga Bradford - teal-lune
- Harvey Duran - harv-d
- Robert Mozzetti - rmozzetti
- Mikey Voss - VossProject

## Tech
- Java / JavaFX
- AtlantaFX
- Gradle
- SQLite (JDBC)
